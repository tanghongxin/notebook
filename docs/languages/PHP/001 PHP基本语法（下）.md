## 操作符（运算符）

> 赋值运算符

```
$a = 0;
```

> 增强运算符

```
$a = 0;$a++; // $a === 1
```

> 增强赋值运算符 += -= *= /=

```
$a = 0;$b = 1;$a += $b; // $a === 1
```

> 算术运算符

```
$a = 10, $b = 5;$c = $a++ / $b--+$a--;/*** 等价于** $d = $a / $b; // $d = 2;** $a++; $b++; //$a == 11; $b == 6** $c = $d + $a--;// $c == 13*/$d = ++$b + $a-- % $b++;/*** 等价于** $d = ++$b; // $d == 6;** $e = $a-- % $b++; // $e == 4; $a == 9; $b == 7** $c = $d + $e; // $c == 10;*/
```

> 逻辑运算符

```
$a = 10, $b = 5;$c = $a > 5|| $b > 5; // $c == ture;
```

> 比较运算符

```
$a = 10, $b = 5;$c = $a > $b; // $c == true;
```
