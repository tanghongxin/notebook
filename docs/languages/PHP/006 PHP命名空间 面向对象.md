### 静态变量 / 方法

- 静态属性只能被初始化为文字或常量
- 静态属性不可以由对象通过 -> 操作符来访问
- 静态方式调用一个非静态方法会导致一个 E_STRICT 级别的错误
- 静态方法不需要通过对象即可调用，所以伪变量 $this 在静态方法中不可用
