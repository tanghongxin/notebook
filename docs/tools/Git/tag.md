# Tag

## 添加 tag

```bash
# 添加 tag
git tag {tagName***REMOVED***

# 添加 tag 并填写内容
git tag {tagName***REMOVED*** - a

# 针对某条 commit 添加 tag
git tag -m {tagContent***REMOVED*** {tagName***REMOVED*** {commitHash***REMOVED***
```

## 删除 tag

```bash
# 删除本地 tag
git tag -d {tagName***REMOVED***

# 删除远程 tag
git push -d {remoteName***REMOVED*** {tagName***REMOVED***
```
