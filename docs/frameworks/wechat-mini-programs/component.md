# 基础组件

> 注意：所有组件与属性都小写，以连字符-连接

```
<tagname property="value">
  Content goes here ...
</tagname>
```

属性类型（支持TS?）

- Boolean
- Number
- String
- Aray
- Object
- EventHandler
- Any

公共属性（指令？）

- id // 组件的唯一标识
- class // 组件的样式类
- style // 组件的内联样式
- hidden // 组件是否显示
- data-* // 自定义属性
- bind* / catch* // 组件的事件
